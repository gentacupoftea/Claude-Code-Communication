認証方法

Conea MCP Platform APIでは以下の認証方法をサポートしています：

  1. APIキー認証 - 開発者ダッシュボードで発行されたAPIキーを使用
  2. OAuth 2.0 - ユーザー認証のための標準的なOAuthフロー
  3. JWT認証 - サービス間の認証に使用

APIキーの例

Authorization: Bearer sk_test_51A1B2C3D4E5F6G7H8I9J0K1L2M3N4O5P6Q7R8S9T0U1V2W3X4Y5Z6

認証方法の詳細は認証ガイドを参照してください。
