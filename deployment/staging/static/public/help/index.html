よくある質問

Conea MCP Platformとは何ですか？

Conea MCP Platform（マルチチャネルプラットフォーム）は、複数のECサイトやSaaSアプリケーションを一元管理するためのプラットフォームです。商品、在庫、注文などのデータを一括管理し、業務効率化を実現します。

APIキーはどこで取得できますか？

APIキーは「設定」ページの「API設定」タブから取得できます。「新規APIキーの発行」ボタンをクリックし、キーの名前と有効期限を設定してください。一度生成されたAPIキーは再表示されないため、安全な場所に保存してください。

データの同期頻度を変更するには？

データ同期の頻度は「設定」ページの「データ管理」タブから変更できます。プランによって設定可能な最小同期間隔が異なります：

  • 無料プラン: 24時間ごと
  • ベーシックプラン: 6時間ごと
  • プロフェッショナルプラン: 1時間ごと
  • エンタープライズプラン: 15分ごと

また、手動同期はいつでも実行できます。

複数のアカウントを接続できますか？

はい、プランによって接続可能なアカウント数が異なります：

  • 無料プラン: 1アカウント
  • ベーシックプラン: 5アカウントまで
  • プロフェッショナルプラン: 15アカウントまで
  • エンタープライズプラン: 無制限

アカウントの追加は「接続」ページから行えます。
