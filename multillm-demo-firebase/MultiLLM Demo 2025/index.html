📊 API エンドポイント

  • GET /api/llm/health - システムヘルスチェック
  • POST /api/llm/generate - レスポンス生成
  • POST /api/llm/stream - ストリーミングレスポンス
  • GET /api/llm/providers - 利用可能なプロバイダー一覧
  • GET /api/llm/strategies - 利用可能な戦略一覧

🔧 使用方法

  1. 上部の「APIキー設定」セクションで各プロバイダーのAPIキーを入力
  2. 「APIキーを保存」ボタンをクリック
  3. 「ヘルスチェック」ボタンで接続状態を確認
  4. プロンプトを入力して「テスト実行」でMultiLLMシステムをテスト

注意: APIキーはブラウザのローカルストレージに保存され、サーバーには送信されません。
